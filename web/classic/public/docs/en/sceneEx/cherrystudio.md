# Using Wukong API in Cherry Studio

> Cherry Studio is a multi-model AI assistant for iOS, macOS, and Windows. It makes it easy to switch between different LLMs for work, research, and daily use.

### Step 1
Download Cherry Studio from [https://cherry-ai.com](https://cherry-ai.com).

If the download is slow, scroll down on the download page and try another mirror.

### Step 2
Open the settings page from the lower-left corner and configure Cherry Studio as shown below.

You only need two values:
1. **API endpoint**: `https://api.wukong.support`
2. **API key**: Create and copy your key from [My Tokens](https://wukong.support/console/token)

### Knowledge Base Setup

1. Add one embedding model, such as `qwen3-embedding-8b`, `bge-m3`, `text-embedding-3-small`, or `text-embedding-3-large`
2. Add one reranking model, such as `qwen3-reranker-8b` or `bge-reranker-v2-m3`
