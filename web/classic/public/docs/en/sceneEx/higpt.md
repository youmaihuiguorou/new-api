# Using Wukong API in HiGPT

> HiGPT is a beginner-friendly AI chat client. You can finish the setup in just two steps and start using powerful AI services right away.

### Step 1
Open [https://higpt.app](https://higpt.app).

### Step 2
Open the settings page from the lower-left corner and configure it as shown below.

You only need two values:
1. **API endpoint**: `https://api.wukong.support`
2. **API key**: Create and copy your key from [My Tokens](https://wukong.support/console/token)

#### Custom model names
Use the format `model-name@OpenAI`, for example `deepseek-r1-all@OpenAI`.

For the full list of model IDs, see the [Model List](https://wukong.support/modellist).
