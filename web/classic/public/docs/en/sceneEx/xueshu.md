# Using Wukong API in GPT Academic Optimization

> GPT Academic Optimization provides practical interfaces for LLMs such as GPT and GLM, with a strong focus on paper reading, polishing, translation, and writing. It also supports project analysis, plugin extensions, and multi-model parallel querying.

### Step 1
Open [https://xueshu.higpt.app](https://xueshu.higpt.app).

### Step 2
Configure your API key.

**API key**: Create and copy your key from [My Tokens](https://wukong.support/console/token)
