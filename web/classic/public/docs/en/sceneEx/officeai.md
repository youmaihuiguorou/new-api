# Using Wukong API in OfficeAI Assistant

> OfficeAI Assistant is a free AI office tool designed for Microsoft Office and WPS users. It can help with formulas, writing assistance, meeting notes, summaries, polishing, and other common office workflows.

### Step 1
Download OfficeAI Assistant from [https://www.office-ai.cn](https://www.office-ai.cn/).

### Step 2
Create a new Word document on your desktop, open it, and configure the tool as shown below. The same setup method also applies to WPS.

Fill in the following fields:
1. **API endpoint**: `https://api.wukong.support`
2. **API key**: Create and copy your key from [My Tokens](https://wukong.support/console/token)
