# Zizi Animation Plugin Installation Guide

Install the Wukong API plugin package to add stronger video generation and AI image generation capabilities to Zizi Animation.

### Step 1: Locate the plugin directory
Open the installation root directory of Zizi Animation and go to:

> **Path:** `installation-root\_internal\plugins`

Inside this folder you will see `video_plugins` and `image_plugins`.

---

### Step 2: Install the video plugin
1. Open the `video_plugins` folder
2. Download the package: [Video plugin package](/docs/static/video_plugin_wukong.rar)
3. Extract the downloaded archive directly into the `video_plugins` folder

---

### Step 3: Install the image plugin
1. Go back one level and open the `image_plugins` folder
2. Download the package: [Image plugin package](/docs/static/images_plugin_wukong.rar)
3. Extract the downloaded archive directly into the `image_plugins` folder

---

### Step 4: Restart the application
1. Close Zizi Animation completely
2. Start the application again
3. Open the related feature pages and confirm that the video and image-generation modules have been loaded successfully
