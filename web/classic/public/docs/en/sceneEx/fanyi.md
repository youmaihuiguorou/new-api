# Using Wukong API in Immersive Translate

> Immersive Translate is a popular bilingual web translation extension for browsers.

### Step 1
Install the browser extension from [https://immersivetranslate.com](https://immersivetranslate.com).

### Step 2
Open the settings page and configure it as shown below.

You only need two values:
1. **API endpoint**: `https://api.wukong.support/v1/chat/completions`
2. **API key**: Create and copy your key from [My Tokens](https://wukong.support/console/token)
