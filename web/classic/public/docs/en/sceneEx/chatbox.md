# Using Wukong API in Chatbox

> Chatbox AI is a cross-platform AI client and assistant available on Windows, macOS, Android, iOS, Linux, and the web.

### Step 1
Download Chatbox from [https://chatboxai.app](https://chatboxai.app/).

### Step 2
Open the settings page from the lower-left corner and configure Chatbox as shown below.

Fill in the following fields:
1. **API endpoint**: `https://api.wukong.support/v1`
2. **API key**: Create and copy your key from [My Tokens](https://wukong.support/console/token)

#### Custom model names
Enter the model name directly in the model list, for example `gpt-4o` or `deepseek-r1-all`.

For the full list of supported model IDs, see the [Model List](https://wukong.support/modellist).
