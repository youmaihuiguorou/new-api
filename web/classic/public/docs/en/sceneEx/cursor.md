# Using Wukong API in Cursor

> Cursor is an AI-first code editor that can apply model-generated code with a single click and update classes or functions directly from prompts.

### Step 1
Download Cursor from [https://www.cursor.com](https://www.cursor.com/).

### Step 2
Open the settings page from the upper-right corner and configure Cursor as shown below.

You only need two values:
1. **OpenAI Base URL**: `https://api.wukong.support/v1`
2. **API key**: Create and copy your key from [My Tokens](https://wukong.support/console/token)

#### Custom model names
Enter the model name directly in the model list, such as `gpt-4o` or `deepseek-r1-all`.

For the full list of available model IDs, see the [Model List](https://wukong.support/modellist).
