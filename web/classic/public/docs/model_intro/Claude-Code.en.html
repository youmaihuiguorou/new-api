What is Claude Code?

Claude Code It is a command-line AI programming assistant officially released by Anthropic. It can complete various development tasks such as code generation, code review, bug repair, and document generation through natural language instructions. It supports direct operation of the terminal and understanding of the code base structure, making it suitable for all types of users from novices to professional developers.

✨ The core advantages of Claude Code

  • natural language interaction: No need to learn complex commands, you can operate them in everyday language
  • Multi-language support: Supports mainstream programming languages such as Python, JavaScript, Java, Go, and Rust.
  • File system operations: Can directly read and write files and execute terminal commands
  • Intelligent context understanding: Automatically understand project structure and code relationships
  • Multiple model switching: Support Claude Sonnet 4, Qwen3-Coder, Kimi-K2 and other models
  • Domestic high-speed access: Provide stable domestic transfer service through Wukong API

Applicable scenarios

  • Quickly generate code frameworks and functions
  • Code review and optimization suggestions
  • Bug debugging and bug fixing
  • Write technical documentation and notes
  • Refactoring and code rewriting
  • Learn new programming languages and frameworks

To ensure that Claude Code can run stably, please ensure that your development environment meets the following basic requirements:

Claude Code installation and configuration

🪟 Windows platform installation tutorial

  1. Install Node.js

    Method 1: Official installation package (recommended)

    • visit Node.js official website
    • Download LTS (Long Term Support) version of Windows Installer (.msi file)
    • Double-click the installation package and follow the wizard to complete the installation (just keep the default options)

    Method 2: Use Winget package manager

    # Run in PowerShell or CMD
winget install OpenJS.NodeJS.LTS

    Verify installation

    After the installation is complete, open a new command prompt (CMD) or PowerShell window and run the following command:

    node --version
npm --version

    If the version number is displayed (such as v18.x.x and 8.x.x), the installation is successful.

  2. Install Claude Code CLI

    toAdministrator statusRun the command prompt or PowerShell and execute the following command:

    npm install -g @anthropic-ai/claude-code

    Verify after installation is complete:

    claude --version

    Permissions issue

    If you encounter permission errors, make sure you run the command line as an administrator. Right-click "Command Prompt" or "PowerShell" and select "Run as administrator".

  3. Get Wukong API Key
    • visit Wukong API Console - Token Management
    • Log in to your account (if you don’t have an account, please register first)
    • Click Create new token
    • inSelect groupSelect from the drop-down menu Claude Code dedicated group
    • Fill in the token name (for example: "Windows Claude Code")
    • Clickcreatebutton
    • Copy the generated API key (format: sk-xxxxxxxxxx)

    Safety tips

    The API key is only displayed once, please save it to a safe location now. Do not share the key with others or upload it to a public repository.

  4. Configure Claude Code

    Create configuration file path:%USERPROFILE%\.claude\settings.json

    Use Notepad to create the configuration file:

    # Run in PowerShell
mkdir $env:USERPROFILE\.claude -Force
notepad $env:USERPROFILE\.claude\settings.json

    In the open notepad, paste the following configuration:

    {
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "Paste your API key here",
    "ANTHROPIC_BASE_URL": "https://api.wukong.support"
  }
}

    Close Notepad after saving the file.

    Important note

    • will ANTHROPIC_AUTH_TOKEN Replace the value of with your actual API key
    • Make sure the JSON is well-formed, note the quotes and commas
    • Required when creating a token Claude Code dedicated group
  5. Start and use

    Open a command prompt or PowerShell and go into your project directory:

    cd C:\path\to\your\project
claude

    When running for the first time, Claude Code will perform initial configuration. Just follow the prompts.

    success sign

    If you see the welcome interface and command prompt of Claude Code, the configuration is successful! You can now start using the AI ​​programming assistant.

def quick_sort(arr: list[int]) -> list[int]:
    """
    Quicksort implementation
    
    Args:
        arr: The list of integers to sort
    
    Returns:
        The sorted list
    
    Time complexity: O(n log n) on average
    Space complexity: O(log n) recursive stack space
    """
    if len(arr) <= 1:
        return arr
    
    # Pick a pivot value (use the middle element here)
    pivot = arr[len(arr) // 2]
    
    # Partition into values less than, equal to, and greater than the pivot
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    
    # Recursively sort and combine
    return quick_sort(left) + middle + quick_sort(right)

# Test
if __name__ == "__main__":
    test_arr = [64, 34, 25, 12, 22, 11, 90]
    sorted_arr = quick_sort(test_arr)
    print(f"Original array: {test_arr}")
    print(f"Sorted array: {sorted_arr}")

from typing import Union

def calculate(
    operand_a: Union[int, float], 
    operand_b: Union[int, float], 
    operator: str
) -> Union[int, float]:
    """
    Perform a basic arithmetic operation
    
    Args:
        operand_a: The first operand
        operand_b: The second operand
        operator: The operator ('+', '-', '*', '/')
    
    Returns:
        The calculation result
    
    Raises:
        ValueError: If the operator is invalid
        ZeroDivisionError: If the divisor is 0
    """
    operations = {
        '+': lambda a, b: a + b,
        '-': lambda a, b: a - b,
        '*': lambda a, b: a * b,
        '/': lambda a, b: a / b,
    }
    
    if operator not in operations:
        raise ValueError(f"Unsupported operator: {operator}")
    
    if operator == '/' and operand_b == 0:
        raise ZeroDivisionError("Divisor cannot be zero")
    
    return operations[operator](operand_a, operand_b)

# Usage example
try:
    result = calculate(10, 5, '+')
    print(f"Result: {result}")
except (ValueError, ZeroDivisionError) as e:
    print(f"Error: {e}")

FAQ

Q1: Why do you need to choose the Claude Code dedicated group?

A: Different tools (Claude Code, CodeX, Gemini CLI) have different API calling methods and parameter requirements. The Wukong API provides specially optimized groupings for each tool, ensuring optimal compatibility and performance. Using the wrong grouping can result in:

  • API call fails or returns error
  • Model response format is incompatible
  • Limited functionality or abnormal behavior
  • Billing exception

Therefore, when creating an API token, be sure to select the dedicated group for the corresponding tool.

Q2: What should I do if "command not found" is prompted during installation?

A: This is usually because Node.js or npm is not installed correctly, or the environment variables are not configured. Please follow the following steps to troubleshoot:

  1. Verify Node.js installation： 
    node --version
npm --version
    If the version number is displayed, it means it has been installed.
  2. Reinstall Node.js: If it is not installed or the version is too low, please visit nodejs.org Download the latest LTS version.
  3. Check environment variables（Windows）：
    • Right-click "This PC" → "Properties" → "Advanced System Settings" → "Environment Variables"
    • Confirm PATH Contains the Node.js installation path (such as C:\Program Files\nodejs\）
  4. Restart the terminal: After modifying the environment variables, you need to reopen the command line window.

Q3: After configuration, I cannot connect to the API and it is prompted that the authentication failed?

A: Please check the following points:

  • API key correctness：
    • Confirm the key with sk- Beginning
    • Check whether the copy is complete (no omissions, no extra spaces)
    • Try regenerating a new key
  • Group selection: Must be selected when creating a token Claude Code dedicated group
  • Configuration file format：
    • JSON must be well-formed (quotes, commas, brackets)
    • File encoding is UTF-8
    • The file path is correct (~/.claude/settings.json）
  • network connection: Make sure it is accessible api.wukong.support

Q4: What should I do if there is a prompt about insufficient permissions under Windows?

A: Installing npm packages globally requires administrator privileges. Solution:

  • Method one: Right-click "Command Prompt" or "PowerShell" and select "Run as administrator"
  • Method two: Configure npm to use a user directory (no admin rights required): 
    # Configure the npm global install path
npm config set prefix "%USERPROFILE%\npm-global"

# Add %USERPROFILE%\npm-global to your system PATH

Q9: What should I do if I encounter the error message "rate limit exceeded"?

A: This indicates that the API call frequency limit has been reached. Solution:

  • wait a moment: Usually recovers automatically after 1-2 minutes
  • Check balance: Confirm that the account has sufficient balance
  • Optimized use：
    • Reduce unnecessary consecutive requests
    • Use /clear Clear long conversation history
    • Choose a more economical model (like Haiku instead of Sonnet)
  • Upgrade plan: Contact customer service to learn about higher frequency limit solutions
